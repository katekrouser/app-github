## Test application to practice UI Automation with Playwright
Ngx-Admin Angular 14 application from akveo.com
This is modified and more lightweight version of original application to practice UI Automation with Playwright.
The original repo is here: https://github.com/akveo/ngx-admin

Thanks to bondar-artem for providing the test application, original UI test application code is here: https://github.com/bondar-artem/pw-practice-app

### Steps how to run test UI application locally

* clone the current repository: ```git clone https://github.com/katekrouser/app-github.git ```

* open project folder in terminal: ```cd app-github```

* run command to install all the dependencies and continue in case warnings occur: ```npm install --force```

* run command to start the app at localhost: ```npm start```

* open http://localhost:4200/ in browser 

* to terminate test UI application use shortcut: ```control + C```
